# 计划：启动项目并检查 Web 界面左右布局

## 项目概述
- **项目名称**：AI Summary（AI 批量文本总结助手）
- **技术栈**：Flask + 原生 HTML/CSS/JS
- **启动方式**：`python run.py`，默认监听 `0.0.0.0:5000`
- **布局方式**：左侧边栏（380px 固定宽度）+ 右侧主内容区（flex: 1 自适应）

## 实施步骤

### 步骤 1：安装依赖
- 检查是否已有虚拟环境 `venv`
- 如果没有，创建虚拟环境并激活
- 运行 `pip install -r requirements.txt` 安装 Flask、openai、gunicorn 等依赖

### 步骤 2：启动项目
- 运行 `python run.py` 启动 Flask 开发服务器
- 确认服务在 `http://localhost:5000` 上成功启动
- 在浏览器中打开页面，确认页面可访问

### 步骤 3：检查左右布局
重点检查以下布局问题：

#### 3.1 整体布局结构
- `.container` 的 `display: flex` 是否正常工作
- 左侧 `.sidebar`（380px 固定宽度）和右侧 `.main-content`（flex: 1）是否正确排列
- 两者之间的 `gap: 24px` 间距是否正常

#### 3.2 侧边栏内容
- AI 服务商设置区域是否完整显示
- Prompt 管理区域是否正常
- 回收站折叠区域是否正常
- 侧边栏内容是否溢出或被截断

#### 3.3 主内容区
- "开始处理"按钮是否正常显示
- 处理进度面板布局是否正常
- 结果表格是否正常

#### 3.4 响应式布局
- 在不同窗口宽度下布局是否合理切换
- 1024px 以下是否正确切换为上下布局

#### 3.5 暗色主题
- 暗色主题切换是否影响左右布局

### 步骤 4：修复发现的问题
- 根据检查结果，修复左右布局相关的 CSS/HTML 问题
- 确保修复后布局在主流浏览器中表现一致

## 关键文件
| 文件 | 用途 |
|------|------|
| `run.py` | 应用入口 |
| `app.py` | Flask 主应用 |
| `templates/index.html` | 主页面 HTML |
